# Summary
- 세포를 이해하는 일은 건강과 질병에 필수적이지만, 기존의 고전적인 모델들은 세포를 모델링하고 시뮬레이팅하는데 모자람
- AI-omics의 발전으로 AIVC(AI virtual cell)을 만드는데 좋은 기회가 생김
	- multi-scale, multi-model NN 기반 모델은 분자의 행동, 세포, 조직간의 상태를 모사/시뮬레이션 할 수 있음
- 이 생각으로, 어떻게 AIVC를 설계하고 구축해야 충실한 시뮬레이션, 발견, 연구에 도움을 주면서 세포를 이해하고, 학계간 협력할 수 있을 지에 대한 비전을 제시
